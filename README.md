# Aditya Srivastava Portfolio

Cinematic scrollytelling portfolio for Aditya Srivastava, Software Development Engineer and System Architect.

Live site: https://portfolio-omega-five-q0du9fmhnl.vercel.app/

## Highlights

- Luxury-tech portfolio experience built with Next.js 14, Framer Motion, GSAP, Three.js, and React Three Fiber.
- Scroll-linked system architecture story with a static-first mobile strategy and lazy-loaded WebGL.
- Case-study driven project showcase for AI Mock Interview, SpecPilot, Fresher's Guide, and Text2Mantra.
- Production SEO, Open Graph image, sitemap, robots, manifest, Vercel Analytics, and Speed Insights.
- Mobile-first performance hardening with reduced-motion support and low-power device safeguards.
